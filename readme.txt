Project Name and Description: My project is called "StockTech" and it serves as a stock analysis and stock prediction tool. It's main functons are to predict
stock prices for up to one month in the future, give live stock prices and other data, organize stocks and give percentages of what stock to buy based on what risk
rate one wants and through calculating the sum volatility rate and expected returns, and finally to provide ratio analysis for any given stock and compare the
ratio analyses of stocks against each other.

How to run the project: The project can simply be run through opening the "StockTech" python file and then running the code.

Libraries that need to be installed: This project requires that the numpy, datetime, matplotlib, pandas, pandas datareader, sci-kit learn, requests, and beautifulsoup4
modules are installed.

Shortcut commands: There are no shortcut commands.
